STM32CubeH7RS Firmware Package V1.3.0 / 29-October-2025

Main Changes

  • General updates to fix known defects and enhancements implementation
  • SLA0044 Legal terms updated to latest revision (Rev 6 / October 2025)
  • SLA0048 Legal terms updated to latest revision (Rev 5 / October 2025)
  • Upgrade of STM32 ExtMem Manager library to new version V1.5.0 :
    • Addition of a new CUSTOM driver in External Memory Manager middleware allowing external memory management based on a configuration file, describing all required memory configuration steps (configuration register updates) and memory access characteristics and commands.
    • Addition of a new XIP template project (available on DK board), based on External Memory Manager Custom driver use.
  • HAL ETH driver V1.2.0, included in STM32CubeH7RS V1.2.0, breaks compatibility with previous versions and requires updating application code based on older HAL ETH driver versions:
    • The HAL_ETH_SetMACFilterConfig() API was updated to fix the enable/disable broadcast filter configuration.
    • When porting an application to the new HAL ETH driver, configure the MAC filter as follows:
      • Set BroadcastFilter = DISABLE to allow reception of broadcast packets.
      • Set BroadcastFilter = ENABLE to block reception of broadcast packets.
  • Upgrade CMSIS Core to v5.9.0_fix_620_dsp_v1.10.0_nn_v3.1.0_no_doc: Documentation based on jQuery 1.7.1 removed
  • Upgrade of STM32 ExtMem Loader library to new version V1.1.1
  • Upgrade of USB Device library to new version V2.11.5
  • Upgrade of USB Host library to new version V3.5.4
  • Upgrade of USB USBPD Devices library to new version V1.2.2
  • Upgrade of USB USBPD Core library to new version V5.3.1
  • Upgrade of LWIP library to new version V2.2.1 (ST modified 2025-08-04)
    • Security Advisory SA0039 fixed
  • Upgrade of mcuboot library to new version V1.7.2.38
  • Upgrade of mbed-crypto library to new version V3.6.4 (ST modified 2025-07-01)
  • Upgrade of OpenBootloader library to new version V6.1.2
