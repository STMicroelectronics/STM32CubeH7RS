Release Notes for

STM32Cube JPEG Utilities Drivers

Copyright © 2016-2023 STMicroelectronics

Purpose

This driver provides a set of basic functions to convert JPEG “Minimum Coded Units” blocks (MCU) into/from standard RGB, Grayscale or CMYK pixels. It can be used on top of the HAL JPEG driver. The following APIs are available:

  • JPEG_GetDecodeColorConvertFunc
  • JPEG_GetEncodeColorConvertFun
